# electron-live-cast
利用腾讯WEBRTC结合electron制作的实时音视频
Windows Mac Linux多端通用，可修改制作自己的实时音视频软件

##### 运行

``` bash
git clone https://github.com/nick2834/electron-live-cast

cd electron-vue-cast

yarn (npm install)

# 运行开发模式
yarn run dev (npm run dev)

# 打包安装文件 
yarn run build (npm run build)

```
